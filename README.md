Movie Picker (GUI App)

## 📌 Description
Movie Picker is a Python desktop application that helps users discover movies by genre.  
It fetches real-time data from the MUBI API and allows users to select a genre and get a random movie recommendation, including details and trailer access.

---

## ⚙️ Features
- Fetches movie data from MUBI API
- Extracts and displays available genres
- GUI interface built with Tkinter + ttkbootstrap
- Random movie selection by genre
- Displays:
  - Title
  - Genre
  - Year
  - Duration
  - Description
- Clickable trailer link (opens in browser)

---

## 🛠️ Tech Stack
- Python 3
- Tkinter (GUI)
- ttkbootstrap (UI styling)
- Requests (API calls)
- tqdm (progress tracking)
